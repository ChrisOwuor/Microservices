My Past Projects

Kaybros consulting limited

I crafted a dynamic website for Kaybros Consulting Limited, designed to seamlessly bridge the gap between clients and the firm. Leveraging the power of React.js and the flexibility of Tailwind CSS, I built an engaging platform that not only showcases the comprehensive range of services offered by the company but also provides a user-friendly interface for clients to connect and explore testimonials.

Node Js Authentication

I also developed a robust authentication system using Node.js, fortified by JSON Web Tokens (JWT) for secure user authentication, and seamlessly integrated it with a MongoDB database. To ensure the highest level of data security, I implemented the bcrypt package to encrypt user passwords before storing them. This comprehensive authentication solution not only guarantees the confidentiality of user information but also provides a smooth and efficient experience, allowing users to access the platform's features with confidence while maintaining the utmost privacy of their credentials.

Django Blog Web App

I embarked on a captivating journey within the realm of Django, constructing a dynamic blog web application that showcases my proficiency in harnessing the versatile capabilities of this powerful Python framework. Employing Django's robust features, integrating Django forms for seamless data input, implimenting authorization and authentication and implimenting both PostgreSQL and SQLite databases, allowing for efficient data management and retrieval. Notably, the administrative aspect of the project was streamlined through the utilization of the Django admin site, empowering content management with ease.

FindMe Api

I developed an API for the "Find Me" application using Django, leveraging advanced facial recognition technology to assist in the location of missing individuals. Throughout the course of this project, I embarked on an enlightening journey, delving into a diverse range of domains, including database management, machine learning, computer vision, and the intricate intricacies of API development. In addition to the backend API, I also designed a polished frontend for the "Find Me" application. This frontend seamlessly integrates with the backend, delivering an intuitive and engaging user experience. Significantly, the application includes robust production-level authentication, prioritizing user security and privacy. Furthermore, I harnessed the capabilities of Redux Toolkit to optimize state management, ultimately enhancing the application's overall performance and user interaction.
